# Student Roll No. 5 Digits Explanation
digit 1 : class and group
- 1 : class 1, group 1
- 2 : class 1, group 2
- 3 : class 1, group 3
- 4 : class 2, group 1
- 5 : class 2, group 2
- 6 : class 2, group 3
- 7 : class 3, group 1
- 8 : class 3, group 2
- 9 : class 3, group 3
digit 2-5 : roll number of student in this task (max 9999 students in a task)

# class, group, master roll -> 5 digits:
- class 1, group 1, master roll 1 = 10001
- class 1, group 2, master roll 2 = 20002
- class 1, group 3, master roll 3 = 30003
- class 2, group 1, master roll 4 = 40004
- class 2, group 2, master roll 5 = 50005
- class 2, group 3, master roll 6 = 60006
- class 3, group 1, master roll 7 = 70007
- class 3, group 2, master roll 8 = 80008
- class 3, group 3, master roll 9 = 90009

# how to calculate 5 digits with 8 digits task_id and master roll
